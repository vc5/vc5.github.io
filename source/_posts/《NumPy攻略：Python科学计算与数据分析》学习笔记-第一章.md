安装好IPython后，运行`ipython -pylab`
我们看到一个警告
```python
WARNING: `-pylab` flag has been deprecated.
Use `--matplotlib <backend>` and import pylab manually.
```
So，我们应当用`ipython --matplotlib`
启动ipython，并使用`import pylab`导入相关软件包。
